What's New

Features finalized between Java 21 and 25.

Unnamed Variables & Patterns

Java 22 · JEP 456

Use _ for variables you intentionally don't use. No more ignored, unused, or suppressed warnings. Works in catch blocks, for loops, lambdas, patterns, and switch cases.

Before 
try {
    parseJson(input);
} catch (JsonException ex) {
    return fallback;  // 'ex' never used
}

map.forEach((key, value) ->
    process(value));  // 'key' never used
After 
try {
    parseJson(input);
} catch (JsonException _) {
    return fallback;  // explicitly unused
}

map.forEach((_, value) ->
    process(value));  // explicitly unused

Stream Gatherers

Java 24 · JEP 485

Custom intermediate stream operations. Until now, you were limited to map, filter, flatMap, etc. If you needed a sliding window, a running total, or any stateful transformation mid-pipeline — you were out of luck. gather() fills that gap.

// Sliding window of size 3
List<List<Integer>> windows = Stream.of(1, 2, 3, 4, 5)
    .gather(Gatherers.windowSliding(3))
    .toList();
// [[1,2,3], [2,3,4], [3,4,5]]

// Fixed-size batches
List<List<Order>> batches = orders.stream()
    .gather(Gatherers.windowFixed(100))
    .toList();

// Running scan (like reduce but emits each step)
Stream.of(1, 2, 3, 4)
    .gather(Gatherers.scan(() -> 0, Integer::sum))
    .toList();
// [1, 3, 6, 10]

Flexible Constructor Bodies

Java 25 · JEP 513

You can now write statements before super() or this(). Validate arguments, compute values, initialize fields — all before calling the parent constructor. No more awkward static helper methods just to work around the old restriction.

Before 
public class PositiveAmount extends Amount {
    public PositiveAmount(long value) {
        super(validate(value));
    }

    // static helper just to validate
    private static long validate(long v) {
        if (v <= 0)
            throw new IllegalArgumentException();
        return v;
    }
}
After 
public class PositiveAmount extends Amount {
    public PositiveAmount(long value) {
        if (value <= 0)
            throw new IllegalArgumentException();
        super(value);
    }

    // no static helper needed
}

Compact Source Files & Instance Main

Java 25 · JEP 512

The classic public static void main(String[] args) ceremony is no longer required. You can write a simple program without a class declaration, and main can be an instance method with no parameters. Great for scripting, teaching, and quick prototyping.

Before 
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
After 
void main() {
    println("Hello, World!");
}

// That's the whole file.

Module Import Declarations

Java 25 · JEP 511

Import an entire module with a single statement. All public types from all packages exported by the module become available. Reduces boilerplate in files that use many types from the same module.

Before 
import java.util.List;
import java.util.Map;
import java.util.Optional;
import java.util.stream.Collectors;
import java.io.IOException;
import java.nio.file.Path;
After 
import module java.base;

// That's it. List, Map, Optional,
// Collectors, IOException, Path —
// all available.
